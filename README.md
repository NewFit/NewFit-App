# NewFit App

Flutter 기반 뉴핏 모바일 앱

## Setting up

1. Flutter 설치
2. Git hook 셋업
   ```
   dart run husky install
   ```
3. vscode 설치
4. 윈도우 사용자의 경우 Git bash 설치
   - https://git-scm.com/downloads
5. vscode 터미널 열기
6. flutter 의존성 설치
   ```
   flutter pub get
   ```
7. 앱 실행
   ```
   flutter run <TODO>
   ```
